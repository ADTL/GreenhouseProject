Контроллер теплицы на базе Arduino Mega

оглавление
Контроллер предназначен для автоматического поддержания микроклимата в небольших (до 200 м2) теплицах, путём управления фрамугами для проветривания и поддержания необходимого режима влажности почвы с помощью управления поливом.
Сведения о возможностях контроллера будут написаны позже :)
Внимание! Контроллер не предназначен для автоматизации промышленных тепличных хозяйств!

Комплект поставки

В текущий комплект поставки входит документация по исходным кодам контроллера, плюс сами исходные коды. Все затраты, связанные с приобретением микроконтроллера Arduino Mega 2560, равно как приобретение дополнительных электронных компонентов - лежат на пользователе данного комплекта. Автор и разработчик документации и исходного кода не несёт никакой ответственности, ни прямой, ни косвенной, за возможный ущерб, нанесённый оборудованию пользователя вследствие любых действий (или бездействий), идущих вразрез с принципами и возможностями, изложенными в данной документации. Автор оставляет за собой право в любой момент пересмотреть как комплект поставки, так и лицензию на распространение исходных кодов контроллера.
