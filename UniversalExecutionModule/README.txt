Прошивка для универсального исполнительного модуля, предоставляет
8 слотов выноса для управления с контроллера. Типы привязок: 

  1. Управление окнами
  2. Управление каналом полива
  3. Управление каналом досветки
  4. Управление статусом пина

Прошивка тестировалась под Arduino Uno, но должна без особых помех
работать и под Pro Mini и пр. семейством.
